# Markdown Print Layout

Markdown Print Layout renders markdown as a printable document with extra line spacing and a wide right margin for handwritten comments.

## Usage

1. Upload a `.md` file or paste markdown into the editor.
2. Adjust paper size, font size, line spacing, and right margin.
3. Use the print button to open the browser print dialog.

The tool runs entirely in the browser. It supports common markdown blocks such as headings, paragraphs, lists, blockquotes, fenced code blocks, links, and simple tables. HTML comments are filtered out of the printable output, except when they appear inside fenced code blocks.


## History

I noticed that LLMs are remarkably good at reading handwritten notes, and I can review big documents better by hand than on a screen. So I wanted to print markdown documents with extra space for handwritten comments.
